scraping-bases-de-datos
=======================

Extraccion automatizada de datos desde fuentes web y almacenamiento en bases de datos relacionales, desarrollado en Python con Jupyter Notebook.

Descripcion
-----------
Proyecto que combina tecnicas de web scraping con el manejo de bases de datos relacionales. El notebook implementa la extraccion de datos desde paginas web, su limpieza y transformacion, y su posterior insercion y consulta en una base de datos. Desarrollado con fines academicos para explorar el ciclo completo de recoleccion y persistencia de datos.

Tecnologias
-----------
- Python
- Jupyter Notebook
- BeautifulSoup / requests (scraping)
- SQL / base de datos relacional

Estructura
----------
- book.ipynb    Notebook principal con el flujo completo de scraping y base de datos

Como ejecutar
-------------
1. Instalar dependencias: pip install jupyter requests beautifulsoup4
2. Iniciar Jupyter: jupyter notebook
3. Abrir book.ipynb y ejecutar las celdas en orden
